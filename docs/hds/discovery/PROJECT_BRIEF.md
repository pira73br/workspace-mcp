# Project Brief — Workspace MCP

Generic MCP platform for AI-assisted development. Merciless Janitors Studios is one consumer via optional extension—not the product purpose.

## Pivot

Deleted `merciless-janitors-workspace-mcp`. Created `workspace-mcp` with core/extension architecture.

## Mission

Permanent AI dev environment for any repository type. Deny-by-default. Capability-aware.

## M1 Proof

Core read-only, detection report, audit, path protection. No extension tools.
